# 🏠 PG Life Property Listing

A responsive property listing webpage inspired by the PG Life website. This project showcases a modern UI using HTML5 and CSS3 with Flexbox, hover animations, responsive layouts, and Font Awesome icons.

---

## 📌 Features

- ✅ Responsive design using Media Queries
- ✅ Modern navigation bar
- ✅ Property listing cards
- ✅ Property images with details
- ✅ Rating and wishlist section
- ✅ Rent and location information
- ✅ Hover animations on property cards
- ✅ Responsive footer
- ✅ Clean and user-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries
- Font Awesome

---


## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/PG-Life-Property-Listing.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
